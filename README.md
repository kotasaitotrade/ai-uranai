# 🔮 占いポータル

APIなしのローカルロジックで動く12種の占いポータルアプリです。

## 機能一覧

| タブ | 占い | 内容 |
|------|------|------|
| 💫 | 相性占い | 5軸スコア + レーダーチャートで2人の関係性を分析 |
| ⭐ | 星座占い | 今日の運勢（恋愛・仕事・金運・健康）+ ラッキーカラー |
| 🔢 | 数秘術 | ライフパスナンバー・性格診断・パーソナルイヤー |
| ☯️ | 九星気学 | 本命星・性格・吉方位・今年の運勢 |
| 🐾 | 動物占い | 12動物 × 5サブタイプ = 60種の性格診断 |
| ✍️ | 姓名判断 | 五格（天格・地格・人格・外格・総格）分析 |
| 🌌 | ホロスコープ | 9天体の配置 + 円形天宮図 + アスペクト |
| 🀄 | 四柱推命 | 年柱・月柱・日柱 + 五行バランス |
| 📈 | バイオリズム | 身体/感情/知性の3周期グラフ |
| 🩸 | 血液型 | 性格診断 + 2人の相性スコア |
| 🃏 | タロット | 大アルカナ22枚・1/3/5枚スプレッド |
| ☯ | 易経 | 六十四卦 + 六爻 + 変爻 |
